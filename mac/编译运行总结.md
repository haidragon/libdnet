## 编译命令

所有案例都使用相同的编译命令：
```bash
cc -Wall -O2 -I../../include -o <程序名> <源文件.c> ../../src/.libs/libdnet.a
```

或使用 Makefile：
```bash
cd <test_dir>
make
./<程序名>
```

## 注意事项

1. **权限要求**：某些操作需要root权限：
   - test12 - 接口操作（崩溃可能与此相关）
   - test15 - 数据包捕获
   - test19 - 原始socket
   - test20 - 数据包注入

2. **libdnet的局限性**：
   - 数据包捕获功能有限，建议使用libpcap
   - 某些平台特定的API可能不支持

3. **跨平台**：
   - 所有代码在macOS上测试
   - Linux上可能需要调整（如接口名称en0改为eth0）

## API关键点

### 地址操作
- `addr_pton()` - 字符串转地址
- `addr_ntop()` - 地址转字符串
- `addr_cmp()` - 地址比较

### 路由操作
- `route_open()` / `route_close()` - 打开/关闭路由表
- `route_loop()` - 遍历路由表
- `route_get()` - 查询路由

### 校验和
- `ip_cksum_add()` - 校验和累加
- `ip_cksum_carry()` - 校验和进位

### 数据包构造
- `ip_hdr` - IP头部结构
- `eth_hdr` - 以太网头部结构
- `ip_cksum` - 校验和计算

## 学习建议

1. **从基础开始**：test1-test11覆盖核心功能
2. **逐步深入**：test12-test21展示高级用法
3. **实践为主**：修改代码观察结果
4. **结合其他库**：
   - libpcap：数据包捕获
   - libnet：数据包注入
   - nmap：网络扫描
