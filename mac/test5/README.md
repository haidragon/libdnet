# test5 - TCP 数据包操作示例

## 概述

本示例演示 libdnet 的 TCP 数据包操作功能，包括 TCP 数据包构建、标志位处理、三次握手和四次挥手等。

## 功能说明

### 1. TCP 数据包构建

- `tcp_pack_hdr()` - 构建 TCP 头部

TCP 头部格式：
```
+---+---+---+---+---+---+---+---+
|      源端口                 |
+---+---+---+---+---+---+---+---+
|      目标端口               |
+---+---+---+---+---+---+---+---+
|        序列号               |
+---+---+---+---+---+---+---+---+
|        确认号               |
+---+---+---+---+---+---+---+---+
|HL |保留|      标志         |W  |
+---+---+---+---+---+---+---+---+
|        窗口大小             |
+---+---+---+---+---+---+---+---+
|        校验和               |
+---+---+---+---+---+---+---+---+
|        紧急指针             |
+---+---+---+---+---+---+---+---+
|        选项...              |
+---+---+---+---+---+---+---+---+
```

### 2. TCP 标志位

| 标志 | 值 | 描述 |
|------|-----|------|
| FIN | 0x01 | 结束连接 |
| SYN | 0x02 | 同步序列号 |
| RST | 0x04 | 重置连接 |
| PSH | 0x08 | 推送数据 |
| ACK | 0x10 | 确认号有效 |
| URG | 0x20 | 紧急指针有效 |

### 3. TCP 三次握手

1. 客户端 → 服务器: SYN
2. 服务器 → 客户端: SYN+ACK
3. 客户端 → 服务器: ACK

### 4. TCP 四次挥手

1. 客户端 → 服务器: FIN+ACK
2. 服务器 → 客户端: ACK
3. 服务器 → 客户端: FIN+ACK
4. 客户端 → 服务器: ACK

## 编译与运行

```bash
# 编译
make

# 运行
./tcp_packet

# 或直接运行
make run

# 清理
make clean
```

## 相关 API

| 函数 | 描述 |
|------|------|
| `tcp_pack_hdr()` | 构建 TCP 头部 |
| `tcp_checksum()` | 计算 TCP 校验和 |
| `tcp_open()` | 打开 TCP 套接字 |
| `tcp_close()` | 关闭 TCP 套接字 |
| `tcp_send()` | 发送 TCP 数据包 |
