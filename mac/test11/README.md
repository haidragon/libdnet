# test11 - 随机数生成示例

## 概述
演示 libdnet 的随机数生成功能。

## 功能
1. 生成随机字节
2. 生成 32 位随机数
3. 生成指定范围随机数

## 编译与运行
```bash
make
./random_gen
```

## 示例输出
```
libdnet 随机数生成示例

=== 随机字节 ===
生成的 16 字节随机数:
a1 b2 c3 d4 e5 f6 78 90 ab cd ef 01 23 45 67 89

=== 随机 uint32_t ===
r1: 1234567890 (0x499602d2)
r2: 987654321 (0x3ade68b1)

=== 随机范围 ===
1-100 之间的随机数: 42

=== 随机数操作 ===
  rand_bytes()        - 生成随机字节
  rand_uint32()       - 生成 32 位随机数
  rand_uint32_range() - 生成范围随机数

所有演示完成！
```
