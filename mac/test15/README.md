# libdnet 数据包捕获示例 (test15)

## 功能说明

本示例展示如何使用 libdnet 库捕获网络数据包，包括：
- 打开网络接口进行捕获
- 解析捕获的数据包
- 显示数据包基本信息
- 支持中断信号处理

## 编译方法

```bash
make
```

## 运行方法

**注意**: 数据包捕获需要 root 权限

```bash
sudo make run
```

或者直接运行：

```bash
sudo ./packet_capture
```

## API 说明

### eth_open()
打开以太网接口。

```c
eth_t *eth_open(const char *device);
```

### eth_close()
关闭以太网接口。

```c
int eth_close(eth_t *e);
```

### eth_recv()
接收数据包。

```c
ssize_t eth_recv(eth_t *e, void *buf, size_t len);
```

## 数据包解析

程序会解析以下信息：
1. **以太网头**: 源/目标 MAC 地址，以太网类型
2. **IP 头**: 源/目标 IP 地址，协议类型，TTL
3. **十六进制转储**: 前 64 字节的原始数据

## 注意事项

1. **需要 root 权限**：数据包捕获通常需要管理员权限
2. **网络活动**: 如果网络不活跃，可能需要等待数据包
3. **按 Ctrl+C**: 可以随时中断捕获
4. **限制**: libdnet 的捕获功能有限，建议使用 libpcap 进行高级捕获

## 常见问题

**Q: 为什么无法打开接口?**
A: 可能需要 root 权限或接口名称不正确

**Q: 捕获不到数据包?**
A: 确保网络接口有流量，可以尝试访问网站

**Q: 如何捕获所有数据包?**
A: 修改 `max_packets` 变量或移除捕获限制
